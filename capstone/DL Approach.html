                    'A', 'a', 'J', 'S', 'V',

# Prepare the final container
root = pd.DataFrame()

for instance in data:
    #print('Working with file:', path)
    annotation = wf.rdann(instance, 'atr') #Extracting the annotations here. Returns the annotation object here
    #print(annotation.__dict__)
    #print (pd.DataFrame(annotation.aux_note)[0].value_counts()) #Counts of (N or (P etc. Disambiguation -> https://www.physionet.org/physiobank/database/html/mitdbdir/intro.htm
    record = wf.rdsamp(instance) #Read a WFDB record, and return the physical signals and a few important descriptor fields
    #print("Patient Record :",record)
    info = record[0]
    ecgSignals, classes = [], []
    for i, beat in enumerate(annotation.symbol):
        if beat in Annotations:
            if beat == '/': beat = 'K'
            start = annotation.sample[i-1] - halfQRS
            end = annotation.sample[i-1] + halfQRS
            QRS = info[start : end, :]
            if len(QRS) != 2 * halfQRS: continue
            for channel in range(2): #Two channel ECG
                beatclass = '{}_{}'.format(record[1]['sig_name'][channel], beat) #String of form Channel_beatClass
                ecgSignals.append(QRS[:, channel]) #All X values, only one particular Y channel
                classes.append(beatclass)
                
    new_frame = pd.DataFrame({'qrs_data' : ecgSignals,
                              'qrs_type' : classes})
    # Cumulate
    #Only root.append should be enough
    root = root.append(new_frame)

'''
the array holds the (x) coords to draw the ECG signal
fs if the frequency of the signal. 360 here. That is 360 (X, Y) values recorded in a second.
sig_len are the total (X, Y) values. therefore 65000/360 ~ 180.555 or about 3 minutes ECG recording
units is the recording parameter. mV denotes millivotes unit the electrode.
n_sig denotes the channel lead. This is a 2 lead ECG signal. Their can be even 12 lead ECG too.
The electrodes that are connected to the patient body denotes the number of channels.
Where on the body are these channels connected are known by different names like MLII or V4 etc.
Comments are the actual description of doctors when analyzing that particular ECG.
the F/M in the comment is the gender of the patient. The number preceding it is his/her age.
'''

Let us talk about channels like MLII or V1, V2 since they are confusing.
MLII means modified lead 2. That is, the electrode was connected to the limb of the patient to detect electric signals
